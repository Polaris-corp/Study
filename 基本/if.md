##### ifは条件分岐を行う時に使用する
C#での構文はざっくりと以下のようになる

	if(条件式)
	{
		条件式が正しい時
	}
	else
	{
		条件式が正しくない時
	}

条件式には一般的な意味での条件を書いてあげるイメージで問題ない
-	例：
	1. 私は人であるか？Yes/No
	1. あの鳥は燕か？Yes/No
	1. この数字は10より小さいか？Yes/No

Yes/NoはC#の言葉で言うとtrue/falseと置き換えられる
なので、上の例を置き換えると下のようになる
-	例：
	-	1.私は人であるか？true/false
	-	2.あの鳥は鳩か？true/false
	-	3.この数字は10より小さいか？true/false

一番上のC#の構文に当てはめると下のようになる

>####	1.
		if(私は人であるか？)
		{
			私が人である時
		}
		else
		{
			私が人でない時
		}

>####	2.
		if(あの鳥は鳩か？)
		{
			あの鳥が鳩である時
		}
		else
		{
			あの鳥は鳩でない時
		}

>####	3.
		if(この数字は10より小さいか？)
		{
			この数字が10より小さい時
		}
		else
		{
			この数字が10より小さくない時
		}
